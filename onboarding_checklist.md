# Onboarding Checklist: 50+ Real Testnet Users

Use this checklist and tracking template to coordinate onboarding for at least 50 unique testnet users. This ensures each user executes a valid, state-changing on-chain transaction that is fully logged and verified.

## Onboarding Script / Steps

1. **Freighter Installation**: Guide the user to install the [Freighter Wallet](https://www.freighter.app/) extension.
2. **Network Selection**: Open Freighter, go to Settings, and select **Stellar Testnet**.
3. **Wallet Connection**: Navigate to the live ChainKitty dashboard and click **Connect Wallet**.
4. **Account Funding**: Copy the user's public address and use the direct link in the onboarding wizard to request 10,000 XLM from the Stellar Testnet Friendbot.
5. **Action Verification**:
   - The user must either **Launch a New Savings Circle** OR **Join an Open Savings Circle** via the discovery tab, OR **Contribute XLM** to an active circle.
   - *Note*: Simple wallet connection does NOT count as active usage. A signature and transaction submission is mandatory.
6. **Transaction Logging**: Record the transaction hash, address, and date into the tracking table below.
